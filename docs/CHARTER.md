# Project Charter — Prompt Intelligence Platform

## What this project is
A web app that helps developers and FDE practitioners write 
better Claude prompts by diagnosing failures, rewriting them,
and comparing results side by side.

## Who it's for
- Anthropic FDE trainees learning prompt engineering
- Developers building with Claude API
- Enterprise clients being onboarded onto Claude

## What's in scope
- Prompt quality analyzer (6 dimensions)
- Automatic prompt rewriter
- Side by side response comparison
- Prompt history and pattern tracking

## What's out of scope
- User authentication
- Multi-user collaboration
- Custom model fine-tuning

## Core principles
- Spec before code — always write the spec first
- Simple and usable — every feature explainable in one sentence
- FDE relevant — every feature teaches a Claude concept
